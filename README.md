# Heart Disease Risk Prediction-
This project is a web application that predicts the risk of heart disease using machine learning. Users enter basic health details, and the system analyzes them using Artificial Neural Network (ANN) and Random Forest models.
Project Structure-
Heart_disease/
│── app.py          # Python backend
│── templates/      # HTML files
│── static/         # CSS / JS files
│── models/         # ANN & Random Forest models
│── README.md
Tech Stack -
Python
HTML, CSS, JS,scikit-learn, TensorFlow
ANN & Random Forest
How to Run-
Clone the repository
Install required Python libraries
Run app.py
Open the application in your browser

Purpose-
To provide a simple tool for early risk detection and health awareness.
